# 📖 Blog App <a name="about-project"></a>

> ## **Blog App** :- This blog app would allow users to create, edit, and delete posts. Users would also be able to comment on posts and like posts. The app would also have a user authentication system so that users could create accounts and log in to the app.

### Prerequisites

In order to run this project you need:

- A Computer (MAC or PC)
- Code editor (VSCode,Atom etc...)
- A browser (Chrome,Mozilla,Safari etc...)
- Version Control System (Git and Github)
- Ruby installed: To run a Ruby project, users must have Ruby installed on their system. You can include the minimum required version of Ruby needed to run the project.
- Rails installed: To run a Rails project, users must have Rails installed on their system. You can include the minimum required version of Rails needed to run the project.
- PGSQL installed: To run a PGSQL project, users must have PGSQL installed on their system. You can include the minimum required version of PGSQL needed to run the project.

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:ZewdieMc/blog-app.git
```

## Install

Install this project with:

- Install RubyInstaller [Win](https://rubyinstaller.org/)
- Install Ruby for [macOS](https://www.ruby-lang.org/en/documentation/installation/#homebrew)
- Install Ruby for [Linux](https://www.makeuseof.com/install-ruby-on-linux/)

## Usage

To run a Ruby file in the command line, follow these steps:

Open a command prompt or terminal window on your computer.

Navigate to the directory where the Ruby file is stored using the cd command.

Type the following command to run the Ruby file:

```sh
bundle install
rails server
rails db:migrate
```

### - where filename.rb is the name of the Ruby file you want to run.

## 🔭 Future Features <a name="future-features"></a>
